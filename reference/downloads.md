# Time Series of Daily Downloads of a TeX-Editor

The data represent the number of daily downloads of a TeX-editor between
June 2006 and February 2007. The dataset contains 267 observations. The
data have been provided by Christian Weiss.

## Usage

``` r
downloads
```

## Format

A time series (`ts`) object containing daily data from June 2006 to
February 2007.

## Source

Weiss, C.H. (2008) Thinning operations for modelling time series of
counts – a survey. *Advances in Statistical Analysis*, **92**, 319–341.
