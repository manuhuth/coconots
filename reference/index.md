# Package index

## Simulate

Simulate convolution-closed count time series.

- [`cocoSim()`](https://manuhuth.github.io/coconots/reference/cocoSim.md)
  : Simulation of Count Time Series

## Fit

Fit (Generalized) Poisson integer autoregressive models.

- [`cocoReg()`](https://manuhuth.github.io/coconots/reference/cocoReg.md)
  : Fitting First- and Second Order (G)PAR Models

## Forecast

Probabilistic k-step-ahead forecasts.

- [`predict(`*`<coco>`*`)`](https://manuhuth.github.io/coconots/reference/predict.coco.md)
  : K-Step Ahead Forecast Distributions

## Model assessment

Diagnostics, goodness-of-fit, scoring rules and model comparison.

- [`cocoBoot()`](https://manuhuth.github.io/coconots/reference/cocoBoot.md)
  : Bootstrap Based Model Assessment Procedure
- [`cocoPit()`](https://manuhuth.github.io/coconots/reference/cocoPit.md)
  : Probability Integral Transform Based Model Assessment Procedure
- [`cocoResid()`](https://manuhuth.github.io/coconots/reference/cocoResid.md)
  : Residual Based Model Assessment Procedure
- [`cocoScore()`](https://manuhuth.github.io/coconots/reference/cocoScore.md)
  : Scoring Rule Based Model Assessment Procedure
- [`cocoSoc()`](https://manuhuth.github.io/coconots/reference/cocoSoc.md)
  : Computes Scores for Various Models Maintaining a Common Sample

## Julia backend

Optional Julia backend (automatic differentiation; faster on complex
models).

- [`installJuliaPackages()`](https://manuhuth.github.io/coconots/reference/installJuliaPackages.md)
  : installJuliaPackages
- [`setJuliaSeed()`](https://manuhuth.github.io/coconots/reference/setJuliaSeed.md)
  : Set Seed for julia's Random Number Generator

## Datasets

- [`cuts`](https://manuhuth.github.io/coconots/reference/cuts.md) : Time
  Series of Monthly Counts of Claimants Collecting Wage Loss Benefit
- [`downloads`](https://manuhuth.github.io/coconots/reference/downloads.md)
  : Time Series of Daily Downloads of a TeX-Editor
- [`goldparticle`](https://manuhuth.github.io/coconots/reference/goldparticle.md)
  : Time Series of Gold Particle Counts in a Well-Defined Colloidal
  Solution
